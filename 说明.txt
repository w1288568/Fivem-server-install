三份脚本均已统一为 10 项菜单的管理模式。

# 文件列表

FiveM-server-install-Linux-ARM.sh     # ARM Ubuntu (FEX-Emu)
FiveM-server-install-Linux-x86.sh     # Linux x86_64 通用
FiveM-server-install-Windows.ps1      # Windows AMD64

# 菜单选项

  1.  安装 FiveM 服务端    → 完整安装流程（License Key / Artifacts / server.cfg / 防火墙）
  2.  启动服务端            → Linux: screen 后台启动 / Windows: Start-Process 新窗口
  3.  查看状态 / 最近日志    → 检测进程 + 读取 logs/ 最新 .log 尾 20 行
  4.  重新编辑 server.cfg   → nano (Linux) / notepad (Windows)
  5.  检查防火墙 / 端口放行  → 检测 30120 TCP+UDP + 40120 TCP，可选一键添加缺失规则
  6.  更新服务端核心         → 重新输入 URL → 下载替换 server/（备份回滚机制）
  7.  卸载 FiveM 服务端      → 二次确认 (yes) → 删目录 + 清理 screen 或防火墙规则
  8.  配置 txAdmin 环境      → 选择 MariaDB/MySQL → 安装数据库 → 放行 40120 → 配置指引
  9.  打开 txAdmin 面板      → Windows: 打开浏览器到 localhost:40120 / Linux: 打印外网地址
  0.  退出脚本

# 版本差异

ARM 版：   仅 Ubuntu + FEX-Emu + RootFS （安装步骤多两步）
x86 版：   Ubuntu/Debian/CentOS/Rocky/Alma/Fedora/Arch 通用，原生运行
Windows 版：Windows Server 2016+ / Win10+，PowerShell 5.1+，UTF-16 LE 编码
